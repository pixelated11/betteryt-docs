# BetterYoutube
A multiplatform youtube desktop client built on electron, featuring an adblocker, and return youtube dislike userscript.
***
## Downlading the app
To download the app, with executables and installers already built, go to the repository's releases page [here](https://github.com/pixelated11/better-youtube/releases) and download the file, matching your OS.

## Getting Started
### Dependencies
To build this app, or run it locally, you have to install the dependencies first.
- Node.js v18.0+
- npm v8.0+
### Running the app locally
To run the app locally, clone the repository, and change directory to the repository:
```
git clone https://github.com/pixelated11/better-youtube.git && cd better-youtube
```
Then, install the devDependencies and dependencies for npm:
```
npm install
```
After it's finished, run the app locally:
```
npm run start
```
### Building the app
To build the app, install npm devDependencies and dependencies:
```
npm install
```
Then, run the build script:
**For Windows:**
```
npm run build:windows
```
**For Linux:**
```
npm run build:linux
```
**Both:**
```
npm run build:all
```
The executables will be located in the `dists/` directory. There, you can find the installer, standalone app, and AppImage.
